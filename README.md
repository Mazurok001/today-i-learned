# today-i-learned
commit - збереження (історія зміни файлу);
head - вказівник (теперішнє місцезнаходження);
Репозиторій - це сховище;
Директорія - це "папка";
Система контролю версій (version control system) - git; mercurial; subversion та ін. (використовуються для пошуку та зміни файлів; збереження, історії, приналежності файлів; колективної роботи з файлами);
Команди в git:
ls - виводить список файлів в директорії;
cd - перехід в директорію;
git commit - зафіксувати (зберегти);
ls -a - список всіх (і прихованих) комітів;
cd .. - перехід назад (в попередню директорію);
git status - статус репозиторію (що змінилось);
git init - шлях по якому був ініціалізований репозиторій;
git log - перевірка комітів.
